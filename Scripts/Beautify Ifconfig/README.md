# Beautify ifconfig
Display of available network interfaces next to IP, netmask and flags related to each one with more readability.

## Dependencies

These are the dependencies that are required to run the script.

- `ifconfig`: `net-tools` package is required.
- `awk`
- `sed`
- `grep`
- `tr`

## Usage
`chmod +x bifconfig.sh`

`./bifconfig.sh`
